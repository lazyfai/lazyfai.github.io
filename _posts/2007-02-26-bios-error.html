--- layout: post title: BIOS error... date: '2007-02-26T09:39:00.000+08:00' author: Fai Wong tags: - life modified_time: '2007-02-26T09:48:10.408+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-7730496803822821186 blogger_orig_url: http://lazyfai.blogspot.com/2007/02/bios-error.html --- My motherboard shows nothing even I plugged a VIA SATA adapter into it, seems due to BIOS not support booting from it. In order to make my card can boot up, I found some people has tried to add the VIA SATA rom file into the motherboard BIOS code to make it bootable.
After several research, I downloaded the SATA ROM and also my motherboard's BIOS, AMIBCP for editing BIOS modules, but seems AMIBCP cannot work correctly on my original BIOS code.
Then I tried to use another recommended tool - mmtool, but seems it will report some strange error too (like free space in BIOS = 2xx K which is impossible).
Finally I ignore and then add the SATA ROM into the BIOS... then flash it, of course, it didn't work afterward, now I need to find a shop to reflash the BIOS for me, poor!

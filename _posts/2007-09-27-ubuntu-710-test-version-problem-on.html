--- layout: post title: Ubuntu 7.10 test version problem on automount date: '2007-09-27T23:32:00.000+08:00' author: Fai Wong tags: - linux modified_time: '2007-09-27T23:34:17.865+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-1696714989983790172 blogger_orig_url: http://lazyfai.blogspot.com/2007/09/ubuntu-710-test-version-problem-on.html --- Just helped a friend to find a bug fix in Ubuntu 7.10 test version.
https://bugs.edge.launchpad.net/ubuntu/+source/gparted/+bug/134712

The error is cannot automount removable volume.
The cause is due to gparted, it will temporary disable automount in gnome when it is working.
But if it crashed or being killed, the temporary disable configuration file won't be removed, so automount is totally disable.
To fix, just remove it by hand, simple.

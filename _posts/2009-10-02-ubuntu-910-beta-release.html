--- layout: post title: Ubuntu 9.10 beta release date: '2009-10-02T21:42:00.001+08:00' author: Fai Wong tags: - linux modified_time: '2009-10-02T21:45:50.421+08:00' thumbnail: http://3.bp.blogspot.com/_ZwYFGbHC6Q8/SsYDWs3gVII/AAAAAAAAFiA/VJhiBEJQRt8/s72-c/preview-1.png blogger_id: tag:blogger.com,1999:blog-8503500.post-629807997684546706 blogger_orig_url: http://lazyfai.blogspot.com/2009/10/ubuntu-910-beta-release.html --- Ubuntu 9.10 beta release today, and it also comes with
Netbook Remix, a specially tuned GUI for netbooks like
ASUS Eee PC.

I have installed on my Eee PC 701 but don't have much
test on it yet, first impression is quite smooth and nice.

New features:
- A new menu system for simple layout
- A new and nice splash screen
- Can support ext4 as root filesystem now
- Use Empathy instead of pidgin for IM
- Ubuntu software store for installing software that are
not included on the live CD.

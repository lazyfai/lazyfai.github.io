注意: 此更新光碟現在是測試階段，雖然已經在幾部測試機器中測試過，但也有可能更新失敗(也就是更新不成功啦，沒有什麼害處)，如果有問題請告訴我。

為什麼有這樣的升級光碟?
1. Eee PC 的硬碟分割方法特別
- 開頭的 2.5G 為唯讀，故一般更新不能寫入
- 後面的 1.4G 為可寫入，所有更新、用戶自己的數據都寫在這個分割區
2. 第二個分割區大小有限，只有 1.4G，如果我們一直做更新，用戶可以用的空間就很小了
3. BIOS 中按 F9 的時候，所做的是把第二個分割區格式化，所以之前的更新都會消失
4. 最好的更新方法就是直接把更新寫到第一個分割區了，這樣做有2個好處
- 多些用戶可以用的空間
- F9 還原之後也不會消失不見

更新前的準備：
1. 由於這個更新是用類似 Recovery CD 還原光碟的方法，所以一定要先把所有自己的文件，例如 /home/user/ 中的檔案拷到 USB/SD 上備份
2. 準備華碩的還原光碟（當然，第一次使用時，也可以用 F9 還原）
3. 空白光碟一張，把這個 ISO 檔案 燒成 CD。
4. 燒好後，這張光碟就是我們的升級光碟了！
5. 準備好之後，拿出你的 USB 光碟機，接駁好，開始囉！

步驟:
1. 先用 BIOS 按 F9 ，或者用華碩還原光碟的方法， 把 Eee PC 還原成出廠設定。
這個例子中是用還原光碟

還原中...

還原第一個分割區之後，會把第二個分割區刪去（所以一定要先備份好！）

2. 還原成功之後，按 Enter 重新開機，但先不要進入系統
拿出剛才燒錄好的升級光碟，放進光碟機，在 BIOS 中按 "Esc"，
會出現開機選單。

3. 選擇用 USB 光碟機開機，跟還原光碟一樣，會詢問你是不是要更新你的系統到 XEPC 20080209 版本。

4. 想定了沒有? 想好以後就輸入 "yes"，開始囉！
升級會分幾個階段，先會把跟新版本有衝突的軟體移除，然後把全部軟體都升級好，
再來是安裝額外的，好用的軟體，最後會把系統設定寫好。這個過程一直回答 "y" 可以了。


5. 有些軟體的舊版本跟新版本有衝突，會先移除舊版本。


6. 這些就是華碩幾個月以來的升級軟體，不多吧（跟 XEPC 相比，哈哈)！


7. 安裝升級版本中...


8. 最後是安裝額外的軟體，和修改好系統的設定，完成之後，按 Enter 就會 Reboot 了，
再做一些第一次開機的設定，就可以開始使用這個更新版本的 Eee PC 操作系統了。
看看磁碟空間，還有 1.4G 可以用呢！

這片升級光碟包括了:
華碩的更新: 輸入法、泡泡龍遊戲、防毒、字典、觸控板驅動程式、Skype網上電話。
XEPC 的更新: Pidgin、Firefox、Thunderbird、媒體播放器、影片播放器。
新加入的軟體: Real 解碼器、Wine、超任模擬器、大型電玩模擬器、Pidgin 訊息提示、aMSN、KDE藍牙應用程式。
修正: JAVA 中文顯示、dbus 及藍牙驅動程式。

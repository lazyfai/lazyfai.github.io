--- layout: post title: 'Reminder: how to reset oracle application server password' date: '2009-02-16T15:17:00.001+08:00' author: Fai Wong tags: - work modified_time: '2009-02-16T15:19:51.413+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-2302593866321339663 blogger_orig_url: http://lazyfai.blogspot.com/2009/02/reminder-how-to-reset-oracle.html --- Reset the oc4jadmin password using the following procedure while you are logged in as the user who installed the Oracle Application Server instance:

1. Stop OC4J and the Application Server Control.
$ORACLE_HOME/opmn/bin/opmnctl stopall

2. Locate and open the following file in a text editor:
$ORACLE_HOME/j2ee/home/config/system-jazn-data.xml

3. Locate the line that defines the credentials property for the oc4jadmin user.
jazn.com
.
.

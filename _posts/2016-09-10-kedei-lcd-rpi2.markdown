## For my reference, here is the layout of the Kedei 3.5 inch LCD for Raspberry Pi [Website][kedei-site]

### LCD Panel Pin => Raspberry Pi Pin

* GND => 20 GND
* IRQ => 22 GPIO25
* T_CS => 24 GPIO8
* L_CS => 26 GPIO7
* MOSI => 19 GPIO10
* MISO => 21 GPIO9
* CLK => 23 GPIO11
* GND => 25 GND

[kedei-site]: http://www.kedei.net/

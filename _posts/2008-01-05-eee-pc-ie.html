很多網站現在都不跟 w3c 的標準，使用 firefox 看部分這類型的網站是很麻煩的事。
所以就有人制作了 ies4linux，利用 Linux 上用 wine 跑 Windows 軟體的方法，
把 IE5/6 輕易地安裝起來了，現在在 Eee PC 上也可以使用，只要利用 xepc.org 的軟件庫即可。
加上了 xepc.org 的軟件庫在 sources.list 之後（方法請參考之前的文章）
先用 sudo apt-get install wine cabextract 安裝 wine 跟 cabextract

安裝之後抓下 ies4linux，解壓縮，不過由於系統使用中文時，下載元件會有問題，先修改一下
ies4linux-2.99.0 下面的 lib/functions.sh 檔案，找出
pid=$(wget -q -b -t 1 -T 5 -U "$useragent" -o /dev/null $URL $WGETFLAGS -O "$file" | sed -e 's/[^0-9]//g')
改成
pid=$(LC_ALL=C LANG=C wget -q -b -t 1 -T 5 -U "$useragent" -o /dev/null $URL $WGETFLAGS -O "$file" | sed -e 's/[^0-9]//g')
改好了以後再執行 ./ies4linux --no-gui

ies4linux 正在自動下載需要的元件。

下載好之後就會自動安裝 IE 跟 flash。

安裝之後，以後要使用 IE 時可以打 /home/user/bin/ie6 載入。
IE 就這樣出來了。

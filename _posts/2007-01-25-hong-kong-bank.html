--- layout: post title: Hong Kong Bank 之旅 date: '2007-01-25T21:33:00.000+08:00' author: Fai Wong tags: - work modified_time: '2007-01-25T21:41:13.041+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-7569730483859330747 blogger_orig_url: http://lazyfai.blogspot.com/2007/01/hong-kong-bank.html --- Hong Kong Bank 的人想測試一下公司產品, 但係遇到問題.
佢地有部機要裝個 service, 但係有 http/https proxy先出到街, 理論上有proxy也可以安裝到的, 但係佢地裝極都好似 register 唔到咁(register要https access), 開始時懷疑係因為 proxy set 錯或 set 漏野, 試左好多次都未得, 於是我就上去試喇.
昨天 9點準時到達, 試左安裝, 移除幾次, 點都 register 唔到, 之後用各種方法 check, 最後 check 到係 installation program 的問題, 似乎佢唔理 proxy setting, 所以點都 connect 唔到上 server...
返公司之後開始同美國的 developers 討論, 最後得出的結論和我想的一樣!
今日嘗試用 offline install 的方法, 先安裝好, 再用另外的方法 register 同 activate, 終於成功了!
預算星期一上去整, 希望一次過搞定啦!

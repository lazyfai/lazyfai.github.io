--- layout: post title: "新 Hard Disk 續集" date: '2007-01-26T23:53:00.000+08:00' author: Fai Wong tags: - life modified_time: '2007-01-27T00:18:26.747+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-4150335093938655953 blogger_orig_url: http://lazyfai.blogspot.com/2007/01/hard-disk_26.html --- 昨晚努力地把舊 hard disk 的資料 rsync 到妹的 hard disk ( 新的 SATA 給了她用 ), 今早起身, 發現開不到機, 似乎和 /dev/console 有關. 想了想, 想到原因了, 因為 /dev 內的files 是 udev generate 的, 所以 rsync skip 了 (因是另一 filesystem), 不過因為今朝要準時返到公司, 冇搞到.
下午在公司把 iBook 的 external hard disk 換成原本在家中用的 hard disk, 又 clone 了2個幾鐘, 但我 set 錯了 file system, 後來才知是 hfsx 不是 hfs+, 搞到 Mac on Linux 用唔到. 又再 clone, 6點前搞定~
星期一才試了, 而家 iBook 用 60G external, 大了一倍, 可以玩更多野了~ 好野!
今晚上完堂返屋企, 搞定 /dev/console 等等的基本 device files, 可以開機了! 真係好靜!

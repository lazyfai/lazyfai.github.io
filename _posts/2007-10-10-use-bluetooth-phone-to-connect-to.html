--- layout: post title: Use a bluetooth phone to connect to internet in Linux date: '2007-10-10T22:12:00.000+08:00' author: Fai Wong tags: - linux modified_time: '2007-10-10T22:18:43.139+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-1504454182484592790 blogger_orig_url: http://lazyfai.blogspot.com/2007/10/use-bluetooth-phone-to-connect-to.html --- How to use a Bluetooth phone as modem in Linux

Basically several steps:
1. Kernel must support bluetooth and ppp, wvdail and bluez-utils must be installed
2. Turn on the bluetooth function in the phone
3. Use "hcitool scan" to scan for the phone, get the hardware address
4. Use "rfcomm bind xx:xx:xx:xx:xx:xx" to pair up with the phone
5. Search dailup support by "sdptool search DUN"
6. If it can be searched, put login information in /etc/wvdial.conf
7. Content of wvdial.conf
[Modem0]
Modem = /dev/rfcomm0
Baud = 115200
SetVolume = 0
Dial Command = ATDT
Init1 = ATZ
Init3 = ATM0
FlowControl = crtscts

[Dialer utms]
Username = guest
Password = guest
Phone = *99***1#
Stupid Mode = 1
Inherits = Modem0

8. Use "wvdial utms" to connect to internet through 3G phone.

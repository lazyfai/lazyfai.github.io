用 apt-get 的方法可以參考以前的文章，簡單的方法就是:
1. 開終端機 (Ctrl-Alt-T)
2. sudo -s (prompt 會變紅色)
3. echo “deb http://pic.xepc.org/fai/updates p701 main” >> /etc/apt/sources.list
4. apt-get update
5. apt-get install firefox firefox-locale-zh-tw thunderbird thunderbird-locale-zh-tw
就會更新成為 firefox 2.0.0.11 跟 thunderbird 2.0.0.9 了。

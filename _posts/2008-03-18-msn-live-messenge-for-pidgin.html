--- layout: post title: MSN Live Messenge for Pidgin date: '2008-03-18T21:28:00.002+08:00' author: Fai Wong tags: - linux modified_time: '2008-03-18T21:32:37.369+08:00' blogger_id: tag:blogger.com,1999:blog-8503500.post-7915598341032278421 blogger_orig_url: http://lazyfai.blogspot.com/2008/03/msn-live-messenge-for-pidgin.html --- Just found a new plugin for Pidgin on internet, msn-pecan is a re-implementation of MSN protocol, which should support a newer MSN protocol, from the website it said:

This is a fork of that code with a much faster development process.

Features include:
  • Support for personal messages
  • Server-side storage for display names (private alias)
  • Partial direct connection support
  • Improved network IO
  • Improved error handling
  • Network issues tested with netem
  • GObject usage
Now it is under testing, and see if it can be merged into pidgin package for Eee PC later.

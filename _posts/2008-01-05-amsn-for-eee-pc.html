pic.xepc.org 中的軟件庫已過時，修改 sources.list 改成這句：
deb http://updates.xepc.org/ p701 main

安裝 aMSN for Eee PC:
先修改好 /etc/apt/sources.list 指向 updates.xepc.org
然後用 sudo apt-get install amsn 就可以安裝 aMSN 了。
視像會議功能的使用方法，跟 skype 2.0 beta 一樣，
就是在終端機中，輸入以下幾行，先打開攝像機再開 amsn:
1. sudo -s
2. sudo /sbin/modprobe uvcvideo
3. echo 1 > /proc/acpi/asus/camera
4. exit
5. amsn
然後就可以用視像會議了！

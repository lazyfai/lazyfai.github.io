Eee PC 內置的 pidgin (MSN/ICQ/Googletalk) 軟件，在使用 http 方法連 MSN 時會出問題。
今天我自己替 Eee PC 打包了最新版本的 pidgin 2.2.2, 修正了很多問題。
更新方法如下。
1. 開終端機: Ctrl-Alt-T
2. 修改 /etc/apt/sources.list : sudo nano /etc/apt/sources.list
3. 加入這行到最後一行:
deb http://pic.xepc.org/fai/updates p701 main
4. 按 Ctrl-X 儲存及離開
5. 先刪除原來的 pidgin:
sudo dpkg -r pidgin pidgin-data
6. 再來是更新套件列表和安裝新版:
sudo apt-get update
sudo apt-get install pidgin
7. 檢查一下版本:
sudo dpkg -l pidgin
如果是 2.2.2-1+eeepc 就成功了。
